# 🧪 tinkering-grounds

A personal sandbox and playground for experimenting with code across different languages and technologies. Nothing here is production-ready — it's just a space to tinker, break things, and learn.

## 📁 Structure

| Folder | Description |
|--------|-------------|
| [`javascript-typescript/`](./javascript) | JS playground — browser, Node, frameworks, and whatever else |


## 🗂️ How this works

Each folder is organized by language. Inside each one you'll find self-contained experiments — some are one-off scripts, others are small projects. Nothing is expected to be polished or complete.

- Experiments that grow into something real get their own repo
- `wip/` subfolders inside each language folder are for half-baked ideas
- Commit messages here are allowed to say things like "idk why this works but it does"

## 🚀 Getting started

Clone the repo and dive into whichever folder interests you:

```bash
git clone https://github.com/YOUR_USERNAME/tinkering-grounds.git
cd tinkering-grounds
```

Each experiment has its own instructions (if it needs any).

## ⚠️ Disclaimer

This is a personal playground. Code quality varies wildly. You've been warned.
